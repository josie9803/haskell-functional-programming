# Haskell Functional Programming

This repository contains exercises focused on core functional programming concepts implemented in Haskell.

Course page: https://www.cs.sfu.ca/~miltner/courses/current/383/

---

## Topics Explored

- List processing and recursion
- Higher-order functions (`map`, `fold`, etc.)
- Algebraic data types (product and sum types)
- Purely functional data structures
- Expression generation and evaluation
- Program synthesis via incremental search
- List monad and do-notation

---

## Technologies

- Haskell
- Stack / Cabal
- Purely functional design principles
- Test-driven validation

---

## Run

Using Stack:

```bash
stack build
stack test
stack run
```

Using Cabal:

```bash
cabal build
cabal test
cabal run
```
